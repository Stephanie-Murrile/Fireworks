# AI Fourth of July Fireworks

An interactive Fourth of July celebration experience featuring animated bottle rockets, colorful fireworks, and cursor-controlled sparklers built with HTML5 Canvas and JavaScript.

Watch rockets launch into the night sky, burst into patriotic red, white, and blue explosions, and create your own sparkling trails by moving your mouse across the screen.

## Demo

https://Stephanie-Murrile.github.io/Fireworks

Try interacting with:

* Moving your mouse to create sparkling trails
* Clicking anywhere to trigger instant fireworks
* Watching automatic bottle rocket launches
* Enjoying patriotic red, white, and blue firework displays

## Features

* Animated bottle rocket launches
* Realistic firework explosion particle effects
* Interactive mouse-controlled sparklers
* Click-to-launch fireworks
* Patriotic Fourth of July color palette
* Responsive desktop and mobile support
* Built entirely with HTML5 Canvas
* No frameworks required

## Tech Stack

* HTML5
* CSS3
* JavaScript
* Canvas API
* Particle Animation System
* GitHub
